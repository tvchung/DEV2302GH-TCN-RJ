- component:
    + class commponent
    + functioon component

- props: trao đổi dữ liệu giữa các component 
    -> component cha -> component con -> props(render)
    -> component con -> component cha -> props(event)
- state: quản lý dữ liệu trong component -> chỉ tồn tại trong commponent -> sử dụng với class component
- event form:
    + controlled form 
    + uncontrolled form 
- list key -> render    
    + mảng dữ liệu -> map to -> đối tượng -> hiển
